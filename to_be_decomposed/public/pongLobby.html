Setting up table:

  • • Please choose between the 2 rules, `First to 5` or `First to 10`, the glowing light will tell which rule is active.
  • • Host player will be automatically sit at the player 1's seat, but host can drag and drop the badge icon (including everyone else's icon to each seat).
  • • Guest player may join and sit by clicking join on available seat, and if wants to leave, can drag his/her own badge icon to the drop zone (gray area where the AI badges).
  • • Host may also drag the AI player badge icons to play with AI, and may put the unused players or AI players into the gray area.
  • • Once every seats were assigned, the start game button will be enable to the host , the host must press this button to start the game.

How to play:

  • • The goal of the game is to prevent the ball to pass the team's paddle and try to make the ball into opponent's team end zone. Depends on the rule, which team reach the specified point first will win the match.
  • • Player could use "left arrow key" and "right arrow key" to move the paddle of the same color, the paddle with deflect the ball and prevent you from losing. For the classic mode "up arrow key" and "down arrow key" is also enabled..
  • • Beside the Classic Pong Mode, each player will be given with *ONE* special power, can be used only once a match. To trigger the special, press "space bar" or "Enter Key" to activate it, the effect of each specials are different and will effect the whole opponent's team. The player might not see the effect from the special but rest assure, the effects are there (and effected the AI players as well)
  • • The same Special effect won't stacked, but prolonged. Differenct specials may effect at the same time.
  • • Enjoy the play :)
  • • Update! classic BAF (Brothers Are Fighting) mode will let 2 players play using the same keyboard! (what a terriffic way to play the game!), player 1 will control the paddle1 using `W` and `S` keys, while the player 2 will use the arraw keys as usual!
